Clustering e NMDS delle minacce IUCN

Una lezione pratica di ecologia statistica

Valentina La Morgia

2026-06-08

Obiettivo della Lezione

Raggruppare le specie alpine non per tassonomia, ma per il loro profilo di minaccia.

  1. Esplorazione: Capire la distribuzione del rischio tra le specie.
  2. Distanza: Calcolare la disimilarità (Bray-Curtis).
  3. Clustering: Identificare i gruppi di specie in base alle minacce (PAM).
  4. Ordinamento: Visualizzare i risultati (NMDS).

Come primo step, dobbiamo pulire i dati e selezionare le nostre specie “indicatrici”.

2. Distanze e Algoritmo

Normalizzazione (Min-Max)

Per evitare che una minaccia “domini” le altre solo per la scala numerica.

# ---- Create species x threat matrix ----
data_matrix <- df_selected %>%
  group_by(species, IUCN_Threat_group_desc) %>%
  summarise(rank = sum(rank), .groups = "drop") %>% 
  pivot_wider(names_from = IUCN_Threat_group_desc, values_from = rank, values_fill = 0)

data_matrix_numeric <- as.matrix(data_matrix[,-1])
rownames(data_matrix_numeric) <- data_matrix$species

data_scaled <- apply(data_matrix_numeric, 2, 
                     function(x) (x - min(x)) / (max(x) - min(x)))

Perché Bray-Curtis + PAM (K-medoids)?

In ecologia, non usiamo la distanza Euclidea se abbiamo molti zeri.

  • Euclidea: Considera le “doppie assenze” come somiglianza.
  • Bray-Curtis: Si concentra su ciò che le specie hanno in comune.
dist_bray <- vegdist(data_scaled, method = "bray")
dist_matrix <- dist_bray


Il K-means usa le medie (centroidi). Il PAM usa esemplari reali (medoidi).

Vantaggio

Il PAM è molto più robusto agli outlier che abbiamo visto nel boxplot iniziale.

3. Quanti Cluster?

Metodi Statistici

Valutiamo l’ “Elbow” e la “Silhouette” per scegliere \(k\).

fcl_sil <- fviz_nbclust(as.matrix(dist_bray), pam, method = "silhouette")
plot(fcl_sil)
  • Silhouette: Misura quanto una specie è simile al suo cluster rispetto agli altri.
  • Decisione: Anche se il picco è a 3, sceglieremo k = 2 per una separazione biologica più chiara.

Interpretazione del Grafico

  • Punti: Specie. Se sono vicine, hanno minacce simili.
  • Ellissi: Il “cuore” dei due cluster.
  • Frecce: Direzione delle minacce. Una freccia lunga indica una minaccia che “spiega” bene la differenza tra le specie.

# ---- Fit IUCN threat vectors to NMDS ----
# Load required libraries
library(vegan)
library(ggplot2)
library(ggrepel)
library(grid)  # for unit()

# Fit environmental vectors (threats) to NMDS
envfit_threats <- envfit(nmds_res, data_matrix_numeric, permutations = 999)

# Extract vector scores
threat_vectors <- as.data.frame(scores(envfit_threats, display = "vectors"))
threat_vectors$threat <- rownames(threat_vectors)

# OPTIONAL: filter significant vectors only (p < 0.05)
signif_threats <- as.data.frame(envfit_threats$vectors$pvals)
signif_threats$threat <- rownames(signif_threats)
signif_vectors <- merge(threat_vectors, signif_threats, by = "threat") %>%
  filter(`envfit_threats$vectors$pvals` < 0.05)


# ---- NMDS plot: Rescaled arrows for clarity ----
# Set your desired scaling factor
scaling_factor <- 2
# Apply scaling to the significant threat vectors
signif_vectors_scaled <- signif_vectors %>%
  mutate(
    NMDS1 = NMDS1 * scaling_factor,
    NMDS2 = NMDS2 * scaling_factor
  )
# NMDS plot with longer arrows
ggplot(nmds_points, aes(MDS1, MDS2, color = cluster)) +
  geom_point(size = 3, alpha = 1) +
  stat_ellipse() +
  # Arrows for threats (scaled)
  geom_segment(data = signif_vectors_scaled, aes(x = 0, y = 0, xend = NMDS1, yend = NMDS2),
               arrow = arrow(length = unit(0.3, "cm")), color = "black", size = 0.6) +
  # Threat labels just beyond arrow tips
  geom_text(data = signif_vectors_scaled,
            aes(x = NMDS1 * 1.05, y = NMDS2 * 1.05, label = threat),
            color = "black", size = 3.5, fontface = "italic") +
  theme_minimal() +
  labs(
    title = "NMDS of Species with Scaled IUCN Threat Vectors",
    x = "Dimension 1",
    y = "Dimension 2",
    color = "Cluster"
  )

Conclusione

  1. Abbiamo trasformato dati grezzi IUCN in profili ecologici.
  2. Abbiamo scoperto che le specie alpine si dividono in due gruppi di rischio principali.
  3. Il medoide ci dà una specie “simbolo” per ogni gruppo di minaccia.
